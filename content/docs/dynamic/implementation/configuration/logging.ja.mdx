---
title: ロギング
---

// 0: なし、1: エラーのみ、2: エラー + 警告、3: エラー + 警告 + 情報
Dynamicは、開発者コンソール（検証要素メニュー内）に表示されるログの種類を選択できるオプションを提供します。

## ログなし

完全にログを表示しない場合は、設定で値を `0` に変更してください。

## エラーのみ

コンソールにエラーのみを表示し、警告を無視したい場合は、このレベルが適しています！ 設定で値を `1` に変更してください。

## 決めかねている

エラーと警告の両方を表示したい場合は、設定で値を `2` に変更してください。

## すべての情報

文字通り、エラー + 警告 + 情報が含まれます。設定で値を `3` に設定してください。
