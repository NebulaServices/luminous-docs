## 特徴

- カスタマイズ可能で、簡単に設定できる
- 非常にシンプルに使用可能
- 高いサポート力があり、お気に入りのサイトをサポート：
  - Google（ログインおよびスイートアプリ）
  - Youtube
  - Discord
  - TikTok
  - その他多数
- 驚異的な速さ
- TypeScriptで記述

## 始め方（実行方法）

### 方法1

1. Dynamicをクローンし、ディレクトリを変更します

```bash
git clone https://github,com/NebulaServices/Dynamic.git && cd Dynamic
```

2. Bashスクリプトを実行し、スクリプトの指示に従ってください。

```bash
./bin/start.sh
```

### Method 2

1. Dynamicをクローンして、ディレクトリを変更します。

```bash
git clone https://GitHub.com/NebulaServices/Dynamic.git && cd Dynamic
```

2. 依存関係をインストールします

```bash
npm i
```

3. Dynamicバンドルをビルドします

```bash
npm run build
```

4. サーバーを実行します

```bash
npm start
```
