## 役割管理 - /roles

### /roles add - 特殊なプロパティを持つ新しい役割を追加

このステップはボットを使用するために必要です。Administrator 権限を持つユーザーが最初にこのコマンドを実行する必要があります。

| 設定  | 説明                                                                     | 使用例            | 必須     |
| ----- | ------------------------------------------------------------------------ | ----------------- | -------- |
| Role  | 影響を受ける役割                                                         | 役割:`@Admin`     | はい     |
| Limit | このグループに許可される新しい配布制限                                   | 数字: `10` 1 - 99 | いいえ\* |
| Admin | 役割がボットに対して管理者権限を持つかどうか（設定コマンドの使用に必要） | 真偽値: `True`    | いいえ\* |

\*これらの設定のいずれかが必要です

### /roles remove - ボットから役割を削除

| 設定 | 説明             | 使用例        | 必須 |
| ---- | ---------------- | ------------- | ---- |
| Role | 影響を受ける役割 | 役割:`@Admin` | はい |

### /roles edit - 役割の特殊なプロパティを編集

| 設定  | 説明                                                                     | 使用例              | 必須   |
| ----- | ------------------------------------------------------------------------ | ------------------- | ------ |
| Role  | 影響を受ける役割                                                         | Role:`@Admin`       | はい   |
| Limit | このグループに許可される新しい配布制限                                   | Number: `10` 1 - 99 | いいえ |
| Admin | 役割がボットに対して管理者権限を持つかどうか（設定コマンドの使用に必要） | Boolean: `True`     | いいえ |

\*これらの設定のいずれかが必要です

### /roles list - ボットに追加されたすべての役割を一覧表示

オプションなし。

## グループ管理 - /groups

### /groups create - 新しいドメイングループを作成

このステップは必須です。リンズを追加する前にグループを作成する必要があります。

| 設定  | 説明                                                           | 使用例             | 必須   |
| ----- | -------------------------------------------------------------- | ------------------ | ------ |
| Name  | グループ内部の名前（これがグループとのやり取りに使用されます） | テキスト: nebgroup | はい   |
| Label | グループのボタンに表示されるラベル                             | テキスト: Nebula   | はい   |
| Style | グループのボタンのスタイル                                     | 選択肢: Primary    | はい   |
| Emoji | グループのボタンに表示される絵文字                             | テキスト: 🌌       | いいえ |
| Role  | そのグループから配布するために必要なロール                     | ロール: @Member    | いいえ |

### /groups delete - ドメイングループを削除する

| 設定 | 説明           | 使用例             | 必須 |
| ---- | -------------- | ------------------ | ---- |
| Name | グループの名前 | テキスト: nebgroup | はい |

### /groups edit - ドメイングループを編集する

| 設定  | 説明                                       | 使用例             | 必須   |
| ----- | ------------------------------------------ | ------------------ | ------ |
| Name  | グループの名前                             | テキスト: nebgroup | はい   |
| Label | グループのボタンに表示されるラベル         | テキスト: Nebula   | いいえ |
| Style | グループのボタンのスタイル                 | 選択肢: Primary    | いいえ |
| Emoji | グループのボタンに表示される絵文字         | テキスト: 🌌       | いいえ |
| Role  | そのグループから配布するために必要なロール | ロール: @Member    | いいえ |

# ユーザーのリセット - /reset
