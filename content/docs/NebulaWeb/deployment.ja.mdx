---
title: デプロイメント
---

NebulaWebはNebula ServicesおよびNebula Developer Labsの公式フラッグシップです。NebulaWebは、数千の人気サイトをサポートする、驚くほど洗練された機能的なウェブプロキシです。NebulaWebを使えば、可能性は無限大です。

NebulaWebをデプロイするには、サーバーで以下のコマンドを実行してください:

`npm i -g pnpm tsx`

`git clone https://github.com/NebulaServices/Nebula.git`

`pnpm i`

`npm run bstart`

NebulaWebはポート8080で動作します。
