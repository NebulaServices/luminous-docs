Building

How do I build this?

Clone the repository with git

$ git clone https://github.com/titaniumnetwork-dev/Ultraviolet.git

CD into the directory of Ultraviolet

$ cd Ultraviolet

Install all dependencies with npm

$ npm install

Produce the dist/ directory and compile scripts

$ npm run build
